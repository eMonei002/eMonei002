## A few projects worth a look

- **What My Year Sounded Like** - I pulled my own Spotify streaming history and dug into when I actually listen, which genres show up when I'm working versus relaxing, and how my taste shifted over the year. Built in Python with Pandas, with the final story laid out in a clean set of charts.

- **Where the City Slows Down** - An analysis of public transit delay data to find which routes and times of day cause the most frustration. I used SQL to clean and join the raw feeds, then built a Tableau dashboard so you can filter by line and hour and see the patterns yourself.

- **Why Customers Leave** - A churn analysis on a telecom dataset where I tried to figure out which factors actually predict someone canceling. I went beyond the obvious and found a few smaller signals that mattered more than I expected. Done in Python with Pandas and a simple logistic regression.

- **Movies People Pretend to Like** - An exploratory dive into a film ratings dataset comparing critic scores, audience scores, and box office. I wanted to see where the gaps were biggest and what genres split opinion the most. SQL for the heavy lifting, Matplotlib and Seaborn for the visuals.

- **Air Quality and the Weather** - A small project checking whether air quality in a city tracks with weather conditions like wind and temperature. It started as a quick question and turned into a good lesson on correlation not meaning causation.
